# SpeX - A Web Solution for Visually Impaired People

SpeX is a web-based application designed to assist visually impaired individuals by providing real-time news updates and text-to-speech functionalities. The website allows users to read news articles, listen to them in audio format, and explore useful resources for their day-to-day needs. The primary goal of this website is to ensure accessibility, usability, and a seamless experience for visually impaired users.

## Features

- **Real-time News**: Get the latest news updates using the News API.
- **Text-to-Speech**: Listen to articles using the JavaScript audio library.
- **User-friendly Interface**: Simple, intuitive design for easy navigation.

## Tech Stack

- **HTML**: Used for the structure and layout of the website.
- **CSS**: Applied for styling the website, ensuring accessibility and readability.
- **JavaScript**: Used for the logic behind fetching real-time news, text-to-speech conversion, and interaction with the user.
- **News API**: Fetches the latest news articles in real time from a trusted news source.
- **JavaScript Audio Library**: Converts the displayed text into speech, enabling users to listen to news articles.

## Installation

To run the SpeX website locally:

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/speXweb.git
   ```

2. Navigate to the project directory:

   ```bash
   cd speXweb
   ```

3. Open `index.html` in your browser to view the website.
